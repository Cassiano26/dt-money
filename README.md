A simple financial webpage built using **React** and **TypeScrip**.

## Prerequisites

Ensure you have the following installed on your machine:

- **Node.js**
- **npm** 

## Getting Started
### 1. Install dependencies
```bash
npm install
```
### 2. Run Json-server
```bash
npm run dev:server
```
### 3. Start the application
```bash
npm run dev
```
